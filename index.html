Just Your Type was born out of a desire to reinvent and gamify how designers, like you, find fonts. Playing off the dating app model, this application allows you to quickly save the fonts that are JUST YOUR TYPE and create a library of your favorite typography.

Just Your Type was developed in partnership with Hannah Lee and Ryan Vega. It was built with Materilize CSS Framework, JavaScript, JQuery, and accessing the Google Font and Quote Garden API.

Day Plannerclose

A simple calendar application that allows the user to add events/tasks for each hour of the day. Each hour is color coded based on the current time. The events/tasks added by the user will be available to the user after a page refresh. This app runs in the browser and features dynamically updated HTML and CSS powered by jQuery and Moment.js. It was built with HTML, CSS, Bootstrap, Javascript, JQuery, and Moment.js
